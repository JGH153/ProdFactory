## Redis (Local Development)

RedisInsight runs inside Docker, so `127.0.0.1` resolves to the RedisInsight container itself. Use the Docker service name instead:

- **Host**: `redis`
- **Port**: `6379`

RedisInsight UI: http://localhost:5540
